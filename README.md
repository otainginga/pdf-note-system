# PDF双页阅读笔记系统

一个基于Streamlit的个人PDF阅读笔记系统，采用双页阅读器布局，支持文本选择、富文本笔记（Markdown + 图片）、书签导航和自动部署管理。

## ✨ 功能特性

### 📚 书架管理
- 展示所有已导入的PDF书籍列表
- 支持上传新的PDF文件
- 每本书独立存储，包含书籍信息、最后阅读页码、书签列表
- 支持删除书籍（同时删除所有关联笔记和图片）
- 支持重命名书籍

### 📖 双页阅读器
- **布局**：左页 + 右页 + 右侧笔记面板（比例2:2:1.5）
- **单页模式**：支持切换单页阅读，书籍区域和笔记区域1:1
- **翻页控制**：上一页/下一页按钮，页码滑块，数字输入跳转
- **快捷键**：左/右方向键翻页
- **笔记徽章**：页面右上角显示该页笔记数量

### 📝 笔记系统
- 支持Markdown格式笔记（标题、列表、加粗、斜体、表格、代码块）
- 支持插入图片（拖拽或点击上传）
- 自动记录页码和创建/更新时间
- 笔记面板显示当前页面的笔记
- 鼠标悬停自动展开笔记条目
- 支持编辑和删除笔记

### 🔖 书签系统
- 添加/删除书签
- 书签列表显示在侧边栏（默认隐藏）
- 点击书签快速跳转到对应页面
- 支持书签备注

### 📊 统计功能
- 书籍数量、笔记数量、书签数量统计
- 笔记文件大小、图片文件大小统计

## 🚀 快速开始

### 环境要求
- Python 3.9+
- Streamlit 1.28+
- PyMuPDF (fitz) 1.23+
- Pillow 10.0+
- uv (Python包管理器)

### 安装与运行

**方式一：使用启动脚本（推荐）**
```bash
# 安装uv（首次使用）
pip install uv

# 启动应用
双击 start.bat

# 停止应用
双击 stop.bat
```

**方式二：命令行启动**
```bash
# 安装依赖
uv sync

# 启动应用
uv run streamlit run app.py --server.port 8103 --server.address 0.0.0.0
```

### 访问地址
- 本地地址：http://localhost:8103
- 网络地址：http://your-ip:8103

## 🎯 使用指南

### 上传书籍
1. 点击左侧导航栏的「书架管理」
2. 在"上传新书籍"区域点击"浏览文件"选择PDF文件
3. 可选：自定义书籍名称
4. 点击「上传书籍」按钮

### 开始阅读
1. 在书架中找到要阅读的书籍
2. 点击书籍名称展开详情
3. 点击「开始阅读」按钮

### 添加笔记
1. 在阅读页面点击右侧「添加笔记」标签
2. 输入选中的文本（前15字预览）
3. 填写笔记内容（支持Markdown）
4. 可选：上传图片
5. 点击「保存笔记」按钮

### 添加书签
1. 在阅读页面点击「添加书签」按钮
2. 可选：填写书签备注
3. 点击确认

### 翻页操作
- **点击按钮**：◀ 上一页 / ▶ 下一页
- **方向键**：← 上一页 / → 下一页
- **滑块**：拖动滑块跳转页面
- **输入框**：直接输入页码跳转
